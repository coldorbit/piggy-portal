# Co-bounce Logo Package

Fresh logo design for Co-bounce as a software engineering firm.

## Concept
The mark combines a lowercase **c** and **b** into a kinetic monogram: the outer arc represents cooperation and momentum, the inner loop represents resilient engineering systems, and the bouncing dot represents rapid iteration, delivery, and product energy.

## Included files
- `co-bounce-logo-primary.svg` — primary horizontal logo with tagline
- `co-bounce-logo-primary-no-tagline.svg` — primary horizontal logo without tagline
- `co-bounce-logo-dark.svg` — dark-background lockup
- `co-bounce-logo-icon.svg` — icon-only transparent mark
- `co-bounce-logo-icon-contained.svg` — app/social icon on a white rounded tile
- `co-bounce-logo-icon-dark-contained.svg` — app/social icon on a navy rounded tile
- `co-bounce-logo-monochrome.svg` — one-color logo
- `co-bounce-logo-board.svg` — brand presentation board
- PNG exports for quick usage
- `co-bounce-brand-tokens.json` — color and typography tokens

## Suggested usage
Use the full lockup for website headers, proposals, pitch decks, and invoices. Use the contained icon for favicon, social avatar, GitHub organization, product badges, or mobile app icon. Maintain clearspace around the logo equal to at least the dot diameter.
