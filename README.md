# Co-bounce Website

A simple React/Vite landing page for a remote software engineering and AI automation firm.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in your terminal.

## Customize

- Edit company text in `src/main.jsx`
- Edit colors and layout in `src/styles.css`
- Replace the contact form button with Calendly, Typeform, HubSpot, or your own backend API later
